# Portfolio Profesional - Kevin Daniel González Vargas

Este es mi proyecto final del ciclo de Desarrollo de Aplicaciones Web (DAW). 

Es un portfolio donde muestro mis trabajos y donde la gente puede contactarme. También incluye un panel de administración para poder gestionar los proyectos sin tener que tocar el código.

## Qué tiene el proyecto

- Diseño oscuro con detalles en azul y rosa pastel
- Página de inicio con información sobre mí y mis habilidades
- Sección de proyectos (se cargan desde la base de datos)
- Formulario de contacto que guarda los mensajes
- Panel de administración protegido con login
- CRUD completo de proyectos (crear, ver, editar y eliminar)
- Todo responsive (se ve bien en móvil, tablet y ordenador)

## Tecnologías usadas

- HTML5, CSS3 y JavaScript
- PHP con PDO
- MySQL
- Git para el control de versiones

## Cómo probarlo en local

1. Clona o descarga el proyecto
2. Importa la base de datos que está en la carpeta `sql/portfolio.sql`
3. Arranca XAMPP (o Laragon) y pon la carpeta en htdocs
4. Accede desde el navegador a `public/index.php`

## Credenciales del panel de administración

- Usuario: `admin`
- Contraseña: `admin123`

## Estructura principal

- `public/` → Páginas que ve todo el mundo
- `admin/` → Panel privado para gestionar el contenido
- `includes/conexion.php` → Conexión a la base de datos
